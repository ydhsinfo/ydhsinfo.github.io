프로그래밍 방법

블록 기반

MakeCode Editor

드래그 앤 드롭으로 블록을 조합하여 프로그래밍하는 방식으로, 초보자도 쉽게 시작할 수 있습니다.

텍스트 기반

Python Editor

파이썬(Python) 코드를 직접 작성하여 프로그래밍하는 방식으로, 더 세밀한 제어가 가능합니다.

웹 기반 시뮬레이터를 제공하므로 별도의 소프트웨어 설치 없이 웹 브라우저에서 바로 코딩하고 테스트할 수 있습니다.

마이크로비트 구동 방법

1

microbit.org/ko/ 접속

마이크로비트 공식 웹사이트에 접속한 후, 프로그래밍 시작하기 버튼을 클릭합니다.

2

에디터 선택

MakeCode Editor(블록 기반) 또는 Python Editor(텍스트 기반) 중 원하는 에디터를 선택합니다.

3

새 프로젝트 생성

새 프로젝트를 생성하고 코딩을 시작합니다. 시뮬레이터에서 바로 결과를 확인할 수 있습니다.

4

마이크로비트 연결

USB 케이블로 마이크로비트를 컴퓨터에 연결한 후, 프로그램을 전송합니다.

5

실행 확인

마이크로비트에서 프로그램이 정상적으로 실행되는지 확인합니다.

실습 1: 손뼉을 쳐서 전등 켜고 끄기

손뼉을 쳐서 전등 켜고 끄기

소리 센서와 LED를 활용한 IoT 실습

문제 상황

침대에 누워 있을 때, 일어나지 않고도 전등을 끄거나 켜려면 어떻게 해야 할까요?

입출력 설계

입력 소리 크기
처리 큰 소리 감지
LED 제어
출력 LED 켜기/끄기

준비물

마이크로비트 1대
소리 감지 센서 (내장 마이크)
LED (내장)

알고리즘

1

tag 변수를 선언하고, 초깃값을 0으로 설정

2

기준 소리 크기를 100으로 설정

3

소리 크기가 100보다 크면 tag값을 확인

4

tag=0이면 LED ON, tag를 1로 변경

tag=1이면 LED OFF, tag를 0으로 변경

0.5초 일시 중지 (연속 인식 방지) - 손뼉을 한 번 쳤는데 여러 번 인식되는 것을 방지합니다.

Python 코드

Python 
from microbit import *

# 전등 상태를 저장하는 변수 (0: 꺼짐, 1: 켜짐)
tag = 0

while True:
    if 100 < microphone.sound_level():
        sleep(500)
        if tag == 0:
            display.show(Image('99999:' * 5))
            tag = 1
        else:
            display.clear()
            tag = 0

실행 주의점

  • 직접 마이크로비트에 프로그램을 전송하여 실행하는 것을 권장합니다.
  • 시뮬레이션으로 실행할 경우, 소리 크기 슬라이더를 조절하여 테스트해야 합니다.

실습 2: 음식점의 호출 장치 만들기

음식점의 호출 장치 만들기

라디오 통신과 버튼을 활용한 IoT 실습

문제 상황

음식점에서 직원을 호출하고 싶을 때, 각 테이블에서 버튼을 눌러 직원에게 테이블 번호를 알려주려면 어떻게 해야 할까요?

입출력 설계

입력 버튼 누르기
처리 테이블 번호
라디오 전송
출력
LED 번호 표시
소리 알림

준비물

표시용 마이크로비트 1개
테이블용 마이크로비트 3개
총 마이크로비트 4개 필요 (표시용 1개 + 테이블용 3개)

시스템 구성도

테이블 1 A 버튼 → "1" 송출
테이블 2 A 버튼 → "2" 송출
테이블 3 A 버튼 → "3" 송출
라디오 그룹 1
표시용 마이크로비트 번호 표시 + 딩동 소리

테이블용 알고리즘

1

라디오 그룹을 1로 설정

2

A 버튼을 누르면 자신의 테이블 번호를 라디오로 송출

3

LED에 V(체크) 표시를 보여줌

4

1초 후 LED 화면 삭제

테이블용 Python 코드 (테이블 1 예시)

Python - 테이블용 
from microbit import *
import radio

# 라디오 그룹 1 설정
radio.config(group=1)
radio.on()

while True:
    if button_a.was_pressed():
        radio.send("1")        # 테이블 번호 송출
        display.show(Image.YES) # V 표시
        sleep(1000)
        display.clear()

표시용 알고리즘

1

라디오 그룹을 1로 설정

2

라디오로 수신된 번호를 LED에 출력

3

딩동 소리 재생 (솔 → 미)

4

일정 시간 후 LED 화면 삭제

표시용 Python 코드

Python - 표시용 
from microbit import *
import radio
import music

# 라디오 그룹 1 설정
radio.config(group=1)
radio.on()

while True:
    msg = radio.receive()
    if msg:
        display.show(msg)          # 수신 번호 LED 출력
        music.play(["G4:4", "E4:4"]) # 딩동 소리 (솔, 미)
        sleep(2000)
        display.clear()
