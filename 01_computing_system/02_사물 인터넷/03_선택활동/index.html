선택활동

보행 신호등 변경 시기를 휴대 전화 장치로 알리기

마이크로비트의 라디오 통신을 활용하여 보행 신호등 시스템을 구현해 봅시다

협력 실습
소요시간 약 40분
마이크로비트 2대

학생 정보

활동 제출을 위해 정보를 입력하세요

예: https://docs.google.com/spreadsheets/d/abc123.../edit

문제 상황

보행 신호등이 빨간색으로 바뀌기 전에 숫자 표시 없이 깜빡이기만 해서 건너도 되는지 고민할 때가 많고 위험해지기도 한다.

시각 장애인도 소리로 알 수 있는 방법은 없을까?

1

문제 정의하기

해결할 문제를 구체적으로 정의합니다

보행 신호등이 언제 바뀔지 휴대 전화로 알 수 있게 하자.

입출력 설계

구분 내용
입력
타이머 신호
처리
보행 신호등에서 초록색으로 변한 후 빨간색으로 변하기 10초 전에 휴대 전화 장치에서 숫자가 카운트다운
출력
휴대 전화 장치의 액정에 10부터 1까지 표시, 처음 5초 낮은음, 마지막 5초 높은음
2

장치 구성하기

필요한 장치와 부품을 준비합니다

준비물

마이크로비트 1개

  • 라디오 장치 (내장)
  • LED (내장)
  • 타이머 (내장)

휴대 전화 장치

마이크로비트 1개

  • 라디오 장치 (내장)
  • LED (내장)
  • 스피커 (내장)
3

알고리즘 설계하기

각 장치의 동작 알고리즘을 설계합니다

보행 신호등 알고리즘

20초 동안 'red', 20초 동안 'green' 순서대로 반복한다. 'green' 출력 후 10초 후에 라디오 신호를 송출한다.

1

라디오 그룹 1 설정

2

'green' 출력 후 10,000ms 대기

3

변수에 10 저장, 1초마다 1씩 줄이면서 라디오 신호 송출

4

'red' 출력 후 20,000ms 대기

휴대 전화 장치 알고리즘

라디오 신호를 받으면 10~1 순서대로 출력한다.
10~6: '미'음 500ms  |  5~1: '솔'음 500ms

1

라디오 그룹 1 설정

2

수신한 라디오 신호를 LED에 표시

3

신호가 10 이하 6 이상이면 '미'음, 아니면 '솔'음 500ms 출력

4

프로그래밍하기

마이크로비트 코드를 작성합니다

보행 신호등 장치

micro:bit - 보행 신호등 장치 
// 시작하면 실행
radio.setGroup(1)

// 무한반복
basic.forever(function () {
    // 초록 신호 출력
    basic.showString("green")
    // 10초 대기
    basic.pause(10000)

    // 카운트다운 시작
    let count = 10
    for (let i = 0; i < 10; i++) {
        // 라디오로 count 전송
        radio.sendNumber(count)
        // 1초 대기
        basic.pause(1000)
        // count 1 감소
        count += -1
    }

    // (가) 빈칸을 채우세요
    

    // 빨간 신호 출력
    basic.showString("red")
    // 20초 대기
    basic.pause(20000)
})

(가)에 들어갈 코드를 작성하세요

힌트: 카운트다운이 끝난 후, 빨간 신호를 출력하기 전에 어떤 동작이 필요할까요?

휴대 전화 장치

micro:bit - 휴대 전화 장치 
// 시작하면 실행
radio.setGroup(1)

// 라디오 수신 시 실행
radio.onReceivedNumber(function (receivedNumber) {
    // 수신한 숫자를 LED에 표시
    basic.showNumber(receivedNumber)

    // 조건에 따라 다른 음 출력
    if (receivedNumber >= 6) {
        // '미'음 500ms
        music.playTone(330, 500)
    } else {
        // '솔'음 500ms
        music.playTone(392, 500)
    }

    // (나) 빈칸을 채우세요
    
})

(나)에 들어갈 코드를 작성하세요

실행하기

프로그램을 실행하고 결과를 확인합니다

보행 신호등 장치

  • LED에 'green', 'red' 스크롤 표시
  • 빨간색으로 바뀌기 10초 전에 라디오 신호 전송
  • 1초 간격으로 10~1 카운트다운 신호 전송

휴대 전화 장치

  • 10~1 숫자를 LED에 표시
  • 10~6: '미'음(330Hz) 0.5초
  • 5~1: '솔'음(392Hz) 0.5초

동작 흐름

green 20초
10초 카운트다운
red 20초
