01

네트워크의 구성

유선 네트워크 vs 무선 네트워크

구분 장점 단점
유선
안정적인 연결과 빠른 데이터 전송 속도 물리적인 연결이 필요
무선
물리적인 연결 선을 사용하지 않아 자유로움 비교적 보안에 약함
무선 네트워크 기술: 블루투스, 근거리 무선 통신망, 와이파이, 전자 태그 등
IP 주소 형식: IPv4, IPv6
컴퓨팅 기기에 공유 설정을 하면 파일, 프린터 등을 공유할 수 있다.
클라우드 컴퓨팅을 사용하면 문서 공동 작업, 데이터 공유가 가능하다.
02

사물 인터넷

사물 인터넷: 다양한 사물이 인터넷에 연결되어 일상생활 속에서 사용되는 기술
사물 인터넷 기술: 센서 기술, 저장 기술, 분석 기술, 데이터 보안 기술 등
활용 분야: 스마트 홈, 스마트 교육, 스마트 공장, 스마트 헬스케어 등
마이크로비트의 내장 센서와 라디오 통신 기능을 활용하여 블록형/파이썬 코딩으로 구현 가능

문제 1 객관식

네트워크 구성에 필요한 장비 가운데 하나의 선으로 여러 대의 장비를 연결할 수 있는 장치로는 허브 또는 스위치를 사용한다. 또한 내부 네트워크와 외부 네트워크를 연결하기 위해서는 ( )을(를) 사용한다.

문제 2 객관식

무선 네트워크 기술에 대한 설명이다. ( ) 안에 들어갈 용어를 바르게 짝지은 것은?

(㉠)은 10m 이내에서 무선으로 데이터를 주고받을 수 있는 기술로 무선 이어폰, 인공지능 스피커 등을 연결할 때 주로 사용

(㉡)은 대부분의 무선 네트워크에 사용되는 기술로 AP를 이용해 노트북, 스마트폰 등으로 인터넷 사용 가능

무선 신호를 사용해서 사물의 정보를 인지하는 기술을 (㉢)(이)라고 하며, 하이패스, 교통 카드 등에 활용

문제 3 주관식

사물 인터넷에 대한 설명이다. (가)에 들어갈 용어를 쓰시오.

"사물 인터넷은 생활 속에서 다양한 물건이 인터넷에 연결되어 데이터를 주고받을 수 있는 시스템이다. 사물 주변의 상황을 인식하기 위해 주로 (가)을(를) 사용한다."
문제 4 객관식

[처리 조건]에 맞는 사물 인터넷 장치를 구현하려고 할 때 필요한 마이크로비트의 내장된 출력 장치를 고른 것은?

[처리 조건]

횡단보도 보행 신호등이 언제 빨간색으로 변할지 보행자가 휴대 전화 장치로 알 수 있는 기능. 불빛과 소리를 이용.

ㄱ. 버튼 ㄴ. LED ㄷ. 스피커 ㄹ. 빛센서 ㅁ. 라디오신호 ㅂ. 가속도센서
문제 5 객관식

일상생활에서 일어날 수 있는 다양한 상황 중 사물 인터넷이 적용된 것을 있는 대로 고른 것은?

ㄱ. 스마트폰으로 찍은 내 사진을 친구들에게 전송

ㄴ. 노트북 컴퓨터를 켜면 자동으로 바이러스를 체크하고 치료

ㄷ. 버스가 정류장에 언제 도착할지에 대한 정보를 정류장에 있는 사람들에게 알려줌

ㄹ. 냉장고 안의 음료수 칸에 음료수가 없을 경우 사용자의 휴대 전화에 알리고 주문 사이트 연결

문제 6 서술형

사물 인터넷 의료 분야에 적용되어 원격 의료 등이 가능해진 것을 스마트 의료라고 한다. 이와 같이 사물 인터넷이 적용된 다양한 사회의 형태를 찾아보자.
