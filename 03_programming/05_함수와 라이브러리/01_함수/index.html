함수란 무엇인가?

함수는 특정 기능을 수행하기 위해 명령문을 코드로 묶은 것입니다.

함수의 정의

프로그래밍에서 함수란 특정 기능을 수행하기 위해 명령문을 코드로 묶은 것으로, 반복적인 코드를 줄이고 프로그램을 체계적으로 구성할 수 있게 합니다. 수학에서의 함수 f(x)처럼, 입력(인수)을 받아 처리한 후 결과(반환값)를 돌려줍니다.

지역 변수와 전역 변수

변수가 사용될 수 있는 범위(Scope)에 따라 지역 변수와 전역 변수로 나뉩니다.

1 변수 유효 범위 시각화

전역 범위 (Global)
count = 20
지역 범위 (Local) - func()
count = 10
함수 내부의 count는 지역 변수 (전역 변수와 별개)
프로그램 전체에서 접근 가능한 전역 변수
예제 35
count = 20     # 전역 변수

def func():
    count = 10   # 지역 변수
    print("함수 내부 count:", count)

func()
print("함수 외부 count:", count)

2 global 키워드

함수 내부에서 전역 변수를 변경하려면 global 키워드를 사용합니다.

global 키워드 사용 예시
count = 0

def add(a, b):
    global count   # 전역 변수 사용 선언
    count += 1
    return a + b

print(add(3, 5))   # 8
print(add(10, 20)) # 30
print("호출 횟수:", count) # 2
주의: global 키워드 없이 함수 내부에서 전역 변수와 같은 이름의 변수에 값을 대입하면, 그것은 새로운 지역 변수가 됩니다.

탐색 함수 실습

함수와 전역 변수를 활용하여 순차 탐색과 이진 탐색을 구현합니다.

활동 23

순차 탐색 함수 L_search

활동 23 - 순차 탐색
count = 0
def L_search(a, target):
    global count
    n = len(a)
    for i in range(n):
        count += 1
        if a[i] == target:
            return i
    return -1

활동 24

이진 탐색 함수 B_search

활동 24 - 이진 탐색
count = 0
def B_search(a, target):
    global count
    left, right = 0, len(a) - 1
    while left <= right:
        mid = (left + right) // 2
        count += 1
        if a[mid] == target:
            return mid
        elif a[mid] > target:
            right = mid - 1
        else:
            left = mid + 1
    return -1
