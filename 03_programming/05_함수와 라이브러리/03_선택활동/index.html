Step 3. 데이터 분석하기

pandas를 활용하여 기온 데이터를 분석합니다

코드 1

CSV 파일 불러오기 & DataFrame 생성

import pandas as pd

# CSV 파일 불러오기
df = pd.read_csv("temp.csv")

# 월-일만 추출
df["day"] = df["day"].str[5:]

print(df)

DataFrame 출력 결과

59 rows x 2 columns
index day temp
코드 2

가장 추웠던 날 & 가장 따뜻했던 날

# 가장 추웠던 날
df[df["temp"]==df["temp"].min()]["day"]
# 가장 따뜻했던 날
df[df["temp"]==df["temp"].max()]["day"]

결론

분석 기간(1~2월) 중 가장 추웠던 날은 1월 23일로 기온이 -14.0°C까지 내려갔으며, 가장 따뜻했던 날은 2월 14일로 8.8°C를 기록했습니다.

전체 59일 중 영상 기온인 날은 0일에 불과하여, 1~2월은 대부분 영하의 추운 날씨가 지속되었음을 알 수 있습니다.
