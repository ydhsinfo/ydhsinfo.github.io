모듈

모듈이란?

함수, 변수, 클래스를 모아놓은 파일입니다. 파이썬에서는 하나의 .py 파일이 하나의 모듈이 됩니다.

모듈 불러오기 문법

import 모듈명 또는 import 모듈명 as 별명
사용: 모듈명.함수()

도트(.) 표기법

상위개념.하위개념 — 모듈 안에 포함된 함수나 변수에 접근할 때 점(.)을 사용합니다.

math.sqrt(16)

함수 찾기

dir(모듈명)으로 모듈에 포함된 함수와 변수 목록을 확인할 수 있습니다.

dir(math) → ['ceil', 'floor', 'gcd', 'sqrt', ...]

함수 → 모듈 → 라이브러리 관계

작은 단위에서 큰 단위로 묶어 관리합니다. 각 요소를 클릭해보세요.

각 모듈 영역을 클릭하면 설명이 표시됩니다.

예제 36 — math.gcd() 사용하기

math.gcd(a, b)는 두 수의 최대공약수(Greatest Common Divisor)를 구합니다.

Python예제36.py
import math
result = math.gcd(15, 18)
print(result)  # 3
실행 결과: 3

활동 25 — math.sqrt()와 별명 사용하기

math.sqrt(121) 의 결과: 11.0

Python별명(alias) 사용
import math as m
print(m.sqrt(121))  # 11.0

별명(alias)을 사용하면 긴 모듈명을 짧게 줄여 코드를 간결하게 작성할 수 있습니다. as 키워드를 사용합니다.

활동 26 — random.sample()로 청소 당번 정하기

Python활동26.py
import random
a = int(input("학급 인원수: "))  # 30
b = int(input("청소 인원수: "))  # 8
c = list(range(1, a+1))
print("청소 당번: ", random.sample(c, b))
