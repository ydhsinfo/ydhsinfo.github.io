다차원 데이터 구조

데이터를 행과 열, 여러 차원으로 저장하는 방법

컴퓨터에서 처리하는 자료는 다양한 형태로 존재하므로 특성에 맞는 데이터 구조를 선택해야 합니다. 다차원 데이터 구조의 개념과 파이썬 리스트를 활용하는 방법을 학습합니다.

차원의 개념 1차원 리스트 2차원 리스트 이미지와 RGB

학습 목표

  • 1 다차원 데이터 구조의 개념과 필요성을 설명할 수 있다.
  • 2 1차원 리스트의 생성, 인덱싱, 슬라이싱, 연산을 수행할 수 있다.
  • 3 2차원 리스트를 생성하고 행과 열로 데이터에 접근할 수 있다.

차원의 개념

컴퓨터에서 처리하는 자료는 다양한 형태로 존재하므로 특성에 맞는 데이터 구조를 선택해야 한다. 다차원 데이터 구조는 데이터를 행과 열, 또는 여러 차원으로 저장하는 방법이다.

1차원 리스트

리스트 형식

리스트명 = [값1, 값2, 값3, ...]
리스트명[인덱스]

인덱스는 0부터 시작하여 1씩 증가합니다.

빈 리스트: c = [] / 0으로 초기화: d = [0] * 5

예제 15 한 종류 자료형 리스트
Python
a = [1, 3, 5]
print(a, type(a))
>>> [1, 3, 5] <class 'list'>

셀을 클릭하면 인덱스를 확인할 수 있습니다

예제 16 여러 종류 자료형 리스트
Python

슬라이싱 리스트 부분 추출
Python
a = [1, 3, 5, 7]
print(a[1:3]) # [3, 5]
print(a[1:]) # [3, 5, 7]
print(a[:3]) # [1, 3, 5]
print(a[:]) # [1, 3, 5, 7]

예제 17 리스트 연산
Python
n = [10, 20, 30, 40, 50]
print(n[0]) # 10
print(n[-1]) # 50
n[0] = "hello"
print(n) # ['hello', 20, 30, 40, 50]
n[3] = n[1] + n[2]
print(n) # ['hello', 20, 30, 50, 50]
n[1] = n[2] > n[3]
print(n) # ['hello', False, 30, 50, 50]

리스트 관련 명령어

명령어 설명
.append(값)맨 뒤에 값 추가
.insert(인덱스, 값)원하는 위치에 삽입
.remove(값)일치하는 요소 삭제 (첫 번째)
.sort()오름차순 정렬
.pop()맨 뒤 요소 삭제
len(리스트명)데이터 개수 반환
실습 리스트 명령어 실행 (playlist)
Python
playlist = ['나비야', '작은별', '산토끼', '비행기']
playlist.append('산토끼')
playlist.insert(0, '아기 상어')
playlist.remove('산토끼')
playlist.sort()
playlist.pop()
print(len(playlist))

2차원 리스트 형식

Python
a = [['놀이기구', '키 제한', '탑승 인원', '대기 시간(분)'],
['범퍼카', '110cm 이상', 30, 40],
['롤러코스터', '120cm 이상', 60, 100],
['플룸라이드', '140cm 이상', 8, 120]]
print(a[1]) # ['범퍼카', '110cm 이상', 30, 40]
print(a[2][3]) # 100
