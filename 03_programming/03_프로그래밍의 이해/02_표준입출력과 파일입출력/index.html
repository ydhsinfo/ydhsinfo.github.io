컴퓨터 프로그램은 사용자로부터 데이터를 입력받아 처리하고 결과를 출력하는 단계로 진행된다. 키보드와 모니터를 이용하는 표준입출력과 파일을 읽고 쓰는 파일입출력을 학습한다.

2

파일입출력

파일입출력은 파일 안의 데이터를 읽고 수정하고 저장하는 기능이다. 프로그램에서 파일의 데이터를 직접 다룰 수 있어 매우 유용하다.

빠른 처리

많은 양의 데이터를 빠르게 처리할 수 있다.

데이터 유지

프로그램 종료 후에도 데이터가 파일에 유지된다.

파일입출력 처리 과정

파일 열기
open()
읽기 / 쓰기
read() / write()
파일 닫기
close()

파일 열기 형식

파일 객체 = open('파일명', '접근 방식')

파일 접근 방식

"r"
읽기 모드 (Read)

파일의 내용을 읽어온다. 파일이 존재하지 않으면 오류가 발생한다.

f.read()
"w"
쓰기 모드 (Write)

파일에 내용을 쓴다. 기존 내용은 삭제되고 새로 작성된다.

f.write()
"a"
추가 모드 (Append)

파일 끝에 내용을 추가한다. 기존 내용은 그대로 유지된다.

f.write()
실행 전 (파일 내용)
실행 후 (파일 내용)
예제 14 놀이기구 버킷리스트 파일 처리

놀이기구 버킷리스트가 담긴 텍스트 파일을 읽고, 쓰고, 추가하는 과정을 살펴보자.

버킷리스트.txt 초기 내용

롤러코스터
플룸라이드
바이킹
모노레일
# 파일 읽기 (r 모드) f = open("버킷리스트.txt", "r") data = f.read() print(data) f.close()

여기서 잠깐!

for문을 사용하면 파일의 내용을 한 줄씩 읽어올 수 있다. end=""는 print()의 기본 줄바꿈을 제거한다.

# for문으로 한 줄씩 읽기 f = open("버킷리스트.txt", "r") for row in f: print(row, end="") f.close()

주의

"w" 모드는 기존 내용을 모두 삭제하고 새로운 내용을 작성한다.

# 파일 쓰기 (w 모드) - 기존 내용 삭제 후 새로 작성 f = open("버킷리스트.txt", "w") f.write("회전목마") f.close()
# 파일 추가 (a 모드) data = input("타고 싶은 놀이기구는? ") f = open("버킷리스트.txt", "a") f.write("\n" + data) f.close()

CSV 파일이란?

CSV(Comma-Separated Values)는 쉼표(,)로 데이터를 구분하는 텍스트 파일 형식이다. 엑셀 등 스프레드시트 프로그램에서도 열 수 있으며, 프로그래밍에서 데이터를 저장하고 교환할 때 널리 사용된다.

예시: 강감찬,010-1234-5678,서울

비식별화 처리 프로그램 작성

이름의 두 번째 글자를 'O'로, 전화번호는 뒷 4자리만 남긴다.

import csv # 원본 파일 열기 (읽기) f1 = open("당첨자명단.csv", "r", encoding="utf8") data1 = csv.reader(f1) # 결과 파일 열기 (쓰기) f2 = open("비식별화처리.csv", "w", encoding="utf8") data2 = csv.writer(f2) for row in data1: row[0] = row[0].replace(row[0][1], 'O') # 이름 두번째 글자 → 'O' row[1] = row[1][-4:] # 전화번호 뒷 4자리만 data2.writerows([row]) f1.close() f2.close()

학습 정리

표준입출력

  • input() : 키보드 입력 (문자열 반환)
  • print() : 모니터 출력
  • int(), float() : 형 변환

파일입출력

  • open(파일, 모드) : 파일 열기
  • 모드: "r" 읽기 / "w" 쓰기 / "a" 추가
  • close() : 반드시 파일 닫기
