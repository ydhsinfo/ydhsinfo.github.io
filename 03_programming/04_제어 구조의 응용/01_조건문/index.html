조건문

Conditional Statements

생활 속에서 조건을 고려하여 결정하듯이, 프로그램에서도 조건에 따라 다른 명령을 수행할 수 있습니다. 조건문은 프로그램의 흐름을 제어하는 가장 기본적인 구조입니다.

단순 조건문
중첩 조건문
다중 조건문
1

단순 조건문

(1) if문

조건식이 참(True)일 때만 명령문을 실행합니다.

형식

if 조건식:
    명령문 1
명령문 2
조건식이 참일 때만 명령문 1 실행, 명령문 2는 항상 실행

순서도

예제 19

어린이 보호 구역 속도 체크

속도 체크 시뮬레이터
▶ 속도를 입력하세요...
Python 
speed = int(input())
if speed > 30:
    print("과속입니다.")
print("시속 30km를 유지하세요.")
입력: 34
출력: 과속입니다. / 시속 30km를 유지하세요.

(2) if~else문

조건식이 참이면 명령문 1, 거짓이면 명령문 2를 실행합니다.

형식

if 조건식:
    명령문 1
else:
    명령문 2

중첩 조건문

조건문 안에 또 다른 조건문을 포함하여, 더 세밀한 분기 처리를 할 수 있습니다.

형식

if 조건식 1:
    if 조건식 2:
        명령문 1
    else:
        명령문 2
else:
    명령문 3

순서도

예제 21

도로 종류 + 어린이 보호 구역 제한 속도

a = input("고속 도로/일반 도로? ")
if a == "고속 도로":
    limit = 100
else:
    b = input("어린이 보호 구역(있음/없음)? ")
    if b == "있음":
        limit = 30
    else:
        limit = 50
print("제한 속도는", limit, "km/h")

다중 조건문

여러 개의 조건을 순서대로 검사하여 해당하는 조건의 명령문을 실행합니다.

형식

if 조건식 1:
    명령문 1
elif 조건식 2:
    명령문 2
else:
    명령문 3
