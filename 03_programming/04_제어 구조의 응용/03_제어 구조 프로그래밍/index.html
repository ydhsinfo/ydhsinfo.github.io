예제 33

Up-Down 숫자 맞히기 게임

1부터 10 사이의 숫자를 랜덤으로 생성하고, 사용자가 추측한 숫자와 비교하여 업(Up) 또는 다운(Down) 힌트를 제공하는 게임입니다.

updown_game.py 
import random
num = random.randint(1, 10)
guess = int(input("1부터 10 사이의 숫자 입력: "))
for i in range(10):
    if guess == num:
        print("정답: ", num)
        break
    elif guess < num:
        print(guess, "보다 업!")
    else:
        print(guess, "보다 다운!")
    guess = int(input("다시 숫자 입력: "))

실행 예시 (num=6일 때): 5 입력 → "5보다 업!" → 8 입력 → "8보다 다운!" → 6 입력 → "정답: 6"

예제 34

영어 타자 게임

세계 수도 이름을 랜덤으로 출제하고, 사용자가 정확히 타이핑하면 pass, 틀리면 fail을 표시합니다. 3문제를 풀고 성공률을 계산합니다.

typing_game.py 
import random
capital = ["Beijing", "Seoul", "Kathmandu", "Tokyo", "Paris"]
correct = 0
for i in range(3):
    print("*** problem", i+1, "***")
    a = random.randint(0, 4)
    word = capital[a]
    print(word)
    user = input()
    if user == word:
        print("- pass -")
        correct += 1
    else:
        print("- fail -")
success = round(correct/3, 2)
print("success rate", success)

round() 함수

round(number, ndigits) 함수는 숫자를 지정한 소수점 자릿수로 반올림합니다. Python은 은행가 반올림(Banker's Rounding)을 사용합니다.

round_example.py 
a = 3.141592
print(round(a))        # 3
print(round(a, 3))     # 3.142
print(round(0.5))       # 0 (은행가 반올림)
print(round(1.5))       # 2

은행가 반올림이란? 정확히 0.5인 경우, 가장 가까운 짝수로 반올림합니다. 예를 들어 0.5 → 0, 1.5 → 2, 2.5 → 2, 3.5 → 4가 됩니다.
