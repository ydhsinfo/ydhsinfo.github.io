페이지:

반복문이란?

같은 작업을 여러 번 반복해야 할 때 반복문(loop)을 사용하면 코드를 간결하고 효율적으로 작성할 수 있습니다.

회전목마 20바퀴 운행 예시

반복문 없이
print("1바퀴 운행") print("2바퀴 운행") print("3바퀴 운행") # ... 20줄 반복 ... print("20바퀴 운행")

→ 20줄의 코드 필요!

반복문 사용
for i in range(1, 21): print(f"{i}바퀴 운행")

→ 단 2줄로 해결!

0 / 20 바퀴

1부터 100까지의 합 구하기

반복문 없이 (100줄)
s = 0 s = s + 1 s = s + 2 s = s + 3 # ... 100줄 계속 ... s = s + 100 print(s)
for문 사용 (3줄)
s = 0 for i in range(1, 101): s = s + i print(s) # 결과: 5050

파이썬에서는 for문과 while문 두 가지 반복문을 사용합니다. 반복문을 잘 활용하면 수천, 수만 번의 작업도 몇 줄의 코드로 처리할 수 있습니다.

for문

for문은 반복 횟수가 정해진 경우에 주로 사용하는 반복문입니다. 리스트, 문자열, range() 함수 등과 함께 사용합니다.

(1) 단순 for문

형식 1: 리스트
for 변수 in 리스트: 반복 명령문

리스트의 각 요소를 순서대로 변수에 대입

형식 2: 문자열
for 변수 in 문자열: 반복 명령문

문자열의 각 문자를 순서대로 변수에 대입

형식 3: range()
for 변수 in range(시작, 종료, 증감): 반복 명령문

range()가 생성한 숫자를 순서대로 변수에 대입

range() 함수

range([시작값], 종룟값, [증감값])
시작값: 기본값 0
종룟값: 포함하지 않음
증감값: 기본값 1

예시:

range(0,3,1) = range(0,3) = range(3) → 0, 1, 2

(2) 중첩 for문

for문 안에 또 다른 for문을 넣으면 중첩 for문이 됩니다. 바깥쪽 반복(외부 루프)의 각 반복마다 안쪽 반복(내부 루프)이 처음부터 끝까지 완전히 실행됩니다.

for i in range(2): # 외부 루프: 0, 1 for j in range(3): # 내부 루프: 0, 1, 2 print(i, j)

외부 루프 2회 × 내부 루프 3회 = 총 6회 반복

(1, 2)

반복문의 제어

(1) break문

break문은 반복문을 즉시 종료하고 반복문 바깥으로 빠져나갑니다.

예제 31 | q/Q 입력 시 게임 종료

while True: answer = input("계속하시겠습니까? (q: 종료): ") if answer == "q" or answer == "Q": print("게임을 종료합니다.") break print("게임을 계속합니다...")

(2) continue문

continue문은 현재 반복의 나머지 부분을 건너뛰고 다음 반복으로 진행합니다.

예제 32 | 1~100 중 5의 배수 합 구하기

s = 0 for i in range(1, 101): if i % 5 != 0: continue # 5의 배수가 아니면 건너뛰기 s = s + i print(s) # 결과: 1050
