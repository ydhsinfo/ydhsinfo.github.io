2단계: 프로젝트 진행 계획

1

문제 정의 및 분석 (1차시)

  • 해결하고자 하는 문제가 무엇인지 명확히 정의
  • 어떤 학문 분야의 지식이 필요한지 파악
  • 필요한 데이터와 입력값 식별
  • 기대 출력 결과 정하기
2

알고리즘 설계 (1차시)

  • 순서도(플로차트)로 전체 흐름 설계
  • 필요한 함수와 클래스 목록 작성
  • 데이터 구조(리스트, 딕셔너리 등) 결정
  • 역할 분담 계획 수립
3

코드 구현 및 테스트 (2~3차시)

  • 역할 분담에 따라 코드 작성
  • 다양한 입력 값으로 테스트 진행
  • 오류(버그) 발견 시 수정(디버깅)
  • 코드 합치기(통합) 및 최종 테스트
4

발표 및 공유 (1차시)

  • 프로젝트 소개 자료(PPT) 제작
  • 시연 데모 준비
  • 어려웠던 점과 해결 방법 발표
  • 다른 팀 발표 듣고 피드백 제공

3단계: 예시 — 건강 관리 프로그램

다음은 '건강 관리 시스템' 주제의 예시 코드입니다. 참고하여 자신만의 프로그램을 만들어 보세요.

Python 건강 관리 시스템 예시 
class HealthManager:
    """건강 관리 클래스 - 과학(생물) + 수학 + 체육 융합"""

    def __init__(self, name, weight, height, age, gender):
        self.__name = name
        self.__weight = weight   # kg
        self.__height = height   # cm
        self.__age = age
        self.__gender = gender

    def calc_bmi(self):
        "BMI = 체중(kg) / 키(m)²"
        h_m = self.__height / 100
        return self.__weight / (h_m ** 2)

    def bmi_status(self):
        bmi = self.calc_bmi()
        if bmi < 18.5: return "저체중"
        elif bmi < 23:   return "정상"
        elif bmi < 25:   return "과체중"
        else:             return "비만"

    def calc_bmr(self):
        "기초대사량 (Harris-Benedict 공식)"
        if self.__gender == "남":
            return 88.4 + (13.4 * self.__weight) + (4.8 * self.__height) - (5.7 * self.__age)
        else:
            return 447.6 + (9.2 * self.__weight) + (3.1 * self.__height) - (4.3 * self.__age)

    def recommend_exercise(self):
        bmi_s = self.bmi_status()
        plans = {
            "저체중": ["근력 운동 주 3회", "단백질 섭취 증가", "유산소 운동 최소화"],
            "정상":   ["유산소 운동 주 3~5회", "근력 운동 주 2회", "현재 식단 유지"],
            "과체중": ["유산소 운동 주 5회", "식이 조절", "하루 7,000보 걷기"],
            "비만":   ["전문가 상담 권장", "저강도 유산소 매일", "저칼로리 식단"]
        }
        return plans.get(bmi_s, [])

    def report(self):
        bmi = self.calc_bmi()
        bmr = self.calc_bmr()
        print(f"\n=== {self.__name}님의 건강 리포트 ===")
        print(f"BMI: {bmi:.1f} ({self.bmi_status()})")
        print(f"기초대사량: {bmr:.0f} kcal/일")
        print("권장 운동:")
        for ex in self.recommend_exercise():
            print(f"  • {ex}")

# 사용 예시
user = HealthManager("홍길동", 75, 175, 17, "남")
user.report()
