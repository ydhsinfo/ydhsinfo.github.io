2 수학 분야 문제 해결

수학의 통계, 확률, 수치 계산 등을 프로그래밍으로 처리하면 복잡한 계산을 빠르고 정확하게 수행할 수 있습니다. 파이썬의 math, statistics 모듈을 활용합니다.

예시 1: 학급 성적 통계 분석

Python statistics 모듈 활용 
import statistics

scores = [85, 92, 78, 96, 73, 88, 91, 65, 84, 79]

mean = statistics.mean(scores)          # 평균
median = statistics.median(scores)      # 중앙값
mode = statistics.mode(scores)          # 최빈값
stdev = statistics.stdev(scores)        # 표준편차

print(f"평균: {mean}")        # 83.1
print(f"중앙값: {median}")     # 84.5
print(f"표준편차: {stdev:.2f}") # 8.97

# 등급 분류
for s in scores:
    grade = "A" if s >= 90 else "B" if s >= 80 else "C"
    print(f{s}점: {grade}등급)

예시 2: 몬테카를로 시뮬레이션으로 원주율 추정

Python - 확률로 π 계산
import random
import math

def estimate_pi(n):
    inside = 0
    for _ in range(n):
        x = random.uniform(-1, 1)
        y = random.uniform(-1, 1)
        if x**2 + y**2 <= 1:  # 단위원 내부
            inside += 1
    return 4 * inside / n

print(f"π 추정값 (1만 번): {estimate_pi(10000):.4f}")
print(f"실제 π 값:       {math.pi:.4f}")

3 과학·사회 분야 문제 해결

과학: 자유 낙하 시뮬레이션

Python - 물리 시뮬레이션
# 자유 낙하: h = 1/2 × g × t²
g = 9.8   # 중력가속도 (m/s²)
h0 = 100  # 초기 높이 (m)
dt = 0.1  # 시간 간격 (s)

t = 0
h = h0
v = 0

print("시간(s) | 높이(m) | 속도(m/s)")
while h > 0:
    print(f"{t:.1f}    | {h:.1f}   | {v:.1f}")
    v += g * dt      # 속도 증가
    h -= v * dt      # 높이 감소
    t += dt

print(f"지면 도달: {t:.1f}초")
# 이론값: t = √(2h/g) ≈ 4.52초

사회: 연령별 인구 분포 분석

Python - 사회 데이터 분석
# 연령별 인구 데이터 분석
population = {
    "0~14세": 612,    # 만 명 단위
    "15~64세": 3567,
    "65세 이상": 901
}

total = sum(population.values())

print(f"총 인구: {total}만 명")
for age, pop in population.items():
    ratio = pop / total * 100
    print(f{age}: {pop}만 명 ({ratio:.1f}%))

# 고령화 지수 = 65세 이상 / 0~14세 × 100
aging_index = population["65세 이상"] / population["0~14세"] * 100
print(f"고령화 지수: {aging_index:.1f}")

확인 문제

프로그래밍과 다른 학문의 융합에 대한 설명으로 옳지 않은 것은?

① 통계 모듈을 사용하면 수학의 평균, 중앙값 등을 쉽게 계산할 수 있다.
② 물리 공식을 코드로 구현하면 다양한 조건의 시뮬레이션이 가능하다.
③ 창의 융합 프로그래밍은 프로그래밍 언어 하나만 완벽히 알면 충분하다.
④ 사회 데이터를 분석하여 인구 구조 변화 추이를 파악할 수 있다.
