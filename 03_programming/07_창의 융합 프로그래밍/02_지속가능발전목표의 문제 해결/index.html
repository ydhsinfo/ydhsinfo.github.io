지속가능발전목표(SDGs, Sustainable Development Goals)는 2015년 UN이 채택한 2030년까지의 국제 사회 공동 목표로, 17개의 목표와 169개의 세부 목표로 구성됩니다. 프로그래밍은 이러한 전 지구적 문제 해결에 강력한 도구가 됩니다.

2 SDG 13: 기후 행동 — 탄소 발자국 계산기

SDG 13 기후 행동은 기후변화에 대응하기 위한 긴급 조치를 촉구합니다. 개인의 탄소 발자국을 계산하는 프로그램을 만들어 기후 문제를 인식하고 생활 속 실천 방안을 탐구합니다.

탄소 발자국 계산 프로그램

Python SDG 13 기후행동 
# 탄소 배출량 계산 (단위: kg CO₂)
def calc_carbon_footprint(transport_km, electricity_kwh, meat_days):
    # 배출 계수 (출처: IPCC 기준)
    CAR_FACTOR = 0.21          # 자동차 1km당 0.21 kg CO₂
    ELEC_FACTOR = 0.459         # 전기 1kWh당 0.459 kg CO₂
    MEAT_FACTOR = 3.3           # 육류 1일당 3.3 kg CO₂

    transport_carbon = transport_km * CAR_FACTOR
    electricity_carbon = electricity_kwh * ELEC_FACTOR
    meat_carbon = meat_days * MEAT_FACTOR

    total = transport_carbon + electricity_carbon + meat_carbon

    print("=== 월간 탄소 발자국 분석 ===")
    print(f"교통:    {transport_carbon:.1f} kg CO₂")
    print(f"전기:    {electricity_carbon:.1f} kg CO₂")
    print(f"식단:    {meat_carbon:.1f} kg CO₂")
    print(f"합계:    {total:.1f} kg CO₂")

    # 한국인 평균 월 탄소 배출량: 약 500 kg
    if total < 300:
        print("🟢 우수! 평균보다 훨씬 낮습니다.")
    elif total < 500:
        print("🟡 보통. 조금 더 노력해 보세요.")
    else:
        print("🔴 개선 필요. 생활 습관을 바꿔봐요.")

    return total

# 실행 예시
calc_carbon_footprint(
    transport_km=300,   # 월 300km 운전
    electricity_kwh=280, # 월 280kWh 사용
    meat_days=20        # 월 20일 육류 섭취
)

3 SDG 2: 기아 종식 — 식량 분배 최적화

SDG 2 기아 종식은 2030년까지 모든 형태의 기아와 영양 부족을 종식시키는 것이 목표입니다. 프로그래밍을 통해 식량 필요량 계산과 분배 최적화 알고리즘을 구현합니다.

지역별 식량 필요량 분석 프로그램

Python - SDG 2 기아 종식
# 지역별 식량 현황 분석
regions = {
    "아시아":    {"population": 4700, "food_supply": 2800, "need": 2100},
    "아프리카":  {"population": 1400, "food_supply": 2300, "need": 2100},
    "유럽":     {"population": 748,  "food_supply": 3400, "need": 2100},
    "아메리카": {"population": 1000, "food_supply": 3200, "need": 2100}
}

# 식량 불안 지역 파악
print("=== 식량 불안 현황 분석 ===")
food_insecure = []
for region, data in regions.items():
    surplus = data["food_supply"] - data["need"]
    if surplus < 0:
        food_insecure.append(region)
    status = "⚠️ 부족" if surplus < 0 else "✅ 충분"
    print(f"{region}: {data['food_supply']} kcal/일 {status} ({surplus:+d} kcal)")

# 잉여 식량 재분배 시뮬레이션
total_surplus = sum(
    data["food_supply"] - data["need"]
    for data in regions.values()
    if data["food_supply"] > data["need"]
)
print(f"\n재분배 가능 식량: {total_surplus} kcal/일")

확인 문제

SDGs와 프로그래밍의 연계에 대한 설명으로 가장 적절한 것은?

① 프로그래밍은 SDG 중 IT 분야 목표에만 활용될 수 있다.
② 데이터 분석, 시뮬레이션 등 프로그래밍은 다양한 SDG 문제 해결에 활용된다.
③ SDGs는 선진국만의 목표이므로 개발도상국과는 관계없다.
④ 탄소 발자국 계산기는 SDG 7(클린 에너지)과만 관련이 있다.
