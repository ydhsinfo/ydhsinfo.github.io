6단계 분리 이론

6단계 분리 이론은 미국의 사회학자 스탠리 밀그램이 발표한 이론으로, 지구상의 모든 사람이 서로 6단계 이내로 연결되어 있다고 주장합니다.

예시: 아이유에서 케빈 베이컨까지

아이유
브로커
배두나
클라우드 아틀라스
톰 행크스
아폴로 13
케빈 베이컨
총 3단계로 연결!

1
활동 1: 4명의 친구 관계 예제 풀이

1 친구 관계를 그래프로 표현

입력: A C, A D, D C, C B

입력 데이터

A - C A - D D - C C - B

노드와 간선

노드: A, B, C, D (4명)

간선: 4개의 친구 관계

2 모든 친구 쌍에 대한 최소 친구 단계

BFS(너비 우선 탐색)로 각 쌍 사이의 최단 거리를 구합니다.

A B C D
A × 2 1 1
B 2 × 1 2
C 1 1 × 1
D 1 2 1 ×

3 개인별 친구 관계의 합

A 2 + 1 + 1 = 4
B 2 + 1 + 2 = 5
C 1 + 1 + 1 = 3
D 1 + 2 + 1 = 4

4 결과: 가장 영향력 있는 사람

C

가장 영향력 있는 사람: C

친구 관계의 합: 3 (최솟값)

C는 모든 친구에게 평균적으로 가장 가까이 있으므로, 네트워크에서 가장 영향력이 큰 사람입니다.

2
활동 2: 6명의 친구 관계 직접 풀어보기

문제 분석

현재 상태

6명(A~F)의 친구 관계 데이터가 주어져 있다.

목표 상태

가장 영향력 있는 친구를 찾는다.

요구 사항

그래프와 표로 모델링하여 친구 단계의 합이 최소인 사람을 구한다.

제약 조건

친구 관계는 양방향이며, 최소 단계를 사용한다.

친구 관계 입력

A - B A - D B - F D - F C - F C - E E - F

노드: A, B, C, D, E, F (6명) / 간선: 7개의 친구 관계

1 그래프로 모델링

힌트

  • 직접 연결된 친구는 1단계입니다.
  • 한 명을 거쳐야 하면 2단계입니다.
  • 그래프에서 최단 경로를 찾으세요.
  • 표는 대칭입니다 (A→B = B→A).
