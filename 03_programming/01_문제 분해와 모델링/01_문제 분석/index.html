문제 분석이란?

What is Problem Analysis?

문제 분석의 정의

문제는 우리가 해결해야 할 구체적이고 명확한 요구 사항을 말합니다. 문제 분석이란 문제를 명확하게 이해하고, 주어진 문제의 현재 상태와 요구 사항 및 제약 조건을 정의하기 위해 수행하는 과정입니다.

문제 해결의 첫 번째 단계는 문제를 정확히 이해하는 것입니다. 문제를 제대로 분석하지 않으면 올바른 해결책을 찾기 어렵습니다. 문제 분석을 통해 우리는 문제의 본질을 파악하고, 효율적인 해결 방법을 설계할 수 있는 기초를 마련합니다.

문제 분석의 세 가지 핵심 질문

Three Key Questions for Problem Analysis

문제를 분석할 때 다음 세 가지 질문을 통해 문제를 체계적으로 이해할 수 있습니다.

질문 1

현재 상태와 목표 상태

문제의 현재 상태와 목표 상태는 무엇인가?

지금의 상황과 도달하고자 하는 결과를 명확히 구분합니다.

질문 2

요구 사항

문제의 요구 사항은 무엇인가?

문제 해결을 위해 반드시 충족해야 할 조건을 파악합니다.

질문 3

제약 조건

문제의 제약 조건은 무엇인가?

문제 해결 과정에서 지켜야 할 한계와 규칙을 정의합니다.

예제: 퍼레이드 로봇 문제

Example: Robot Navigation Problem

문제 상황

N×N 크기(N: 1~100)의 정사각형으로 구성된 놀이공원에서 이동 계획에 따라 움직이는 퍼레이드 로봇이 있다. 격자 공간의 크기와 이동 계획이 주어졌을 때, 퍼레이드 로봇이 도착한 최종 좌표 P를 출력하는 프로그램을 만들어 보자.

조건

  • 로봇은 시작 좌표 S(1, 1)에서 출발한다.
  • 이동 명령: L(왼쪽), R(오른쪽), U(위), D(아래) - 한 칸씩 이동
  • 격자 공간을 벗어나는 이동은 무시한다.

문제 분석 적용

1

현재 상태와 목표 상태

현재 상태

로봇이 시작 좌표 S(1, 1)에 있다.

목표 상태

로봇이 도착한 최종 좌표를 출력한다.

2

요구 사항

로봇이 이동 계획에 따라 최종적으로 도착한 좌표를 출력한다.

3

제약 조건

  • 상(U), 하(D), 좌(L), 우(R)로 한 칸씩 이동한다.
  • 격자 공간을 벗어나는 움직임은 무시한다.

1 예시 1

격자 크기: 3
이동 계획:
R R D R D L
출력: (3, 2)

2 예시 2

격자 크기: 4
이동 계획:
D D R R U R
출력: (2, 4)
