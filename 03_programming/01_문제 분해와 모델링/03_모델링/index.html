모델링이란?

What is Modeling?

모델링의 정의

모델링(Modeling)이란 분해한 문제에서 추출한 핵심 요소들을 이해하기 쉽게 표와 트리, 그래프 등으로 단순화하거나 구조화하는 과정을 의미합니다.

모델(Model)이란?

문제 해결에 필요한 핵심 요소들 간의 관계를 추상적으로 표현한 모형을 말합니다.

추상화는 문제가 지닌 복잡성을 단순화하고 중요한 특징을 강조하는 과정으로, 추상화에는 문제 분해와 핵심 요소 추출, 모델링 과정이 모두 포함됩니다.

추상화 = 문제 분해 + 핵심 요소 추출 + 모델링

2 트리 (Tree)

데이터 간 관계를 계층적으로 표현하기에 적합한 구조입니다. 루트 노드에서 하위 노드로 간선이 존재하고, 사이클은 허용하지 않습니다.

노드 (Node)

데이터 항목

간선 (Edge)

관계를 선으로 표현

사이클 (Cycle)

순환 구조 (트리에서 불허)

컴퓨터 폴더 구조 조직도 동물의 분류

예시: 동물 분류 트리

노드를 클릭하면 정보를 확인할 수 있습니다.

3 그래프 (Graph)

데이터 간 연결 관계를 노드와 간선으로 표현하는 구조입니다. 트리와 달리 사이클을 허용하며, 화살표로 방향성을 표현하거나 간선 위에 가중치를 표시할 수 있습니다.

트리
  • 계층적 구조
  • 사이클 불허
  • 하나의 루트 노드
그래프
  • 자유로운 연결 구조
  • 사이클 허용
  • 방향성/가중치 표현 가능
항공 노선도 지하철 노선도 인물 관계도

예시: 항공 노선도

노드(도시)를 클릭하면 해당 도시와 연결된 노선이 강조됩니다.
