Key Concepts Summary

01 디지털 사회와 진로

디지털 기술

정보를 처리, 저장, 전송하는 장치를 연구하고 개발하는 기술입니다.

사회 변화

디지털 기술은 개인의 삶과 사회 전반에 활용되며 사회 변화를 이끌고 있습니다.

진로 탐색

디지털 기술이 활용된 진로와 디지털 기술 분야의 진로를 폭넓게 탐색할 수 있습니다.

02 정보 보호와 보안

정보의 구분

보호해야 할 정보와 공유해야 할 정보를 올바르게 구분하는 것이 중요합니다.

정보 보안 위협 유형

위조

송신자를 속여 허위 데이터를 보내는 행위

변조

원본 데이터를 무단으로 수정하는 행위

접근과 유출

권한 없이 정보에 접근하여 유출하는 행위

훼손과 파괴

정보를 의도적으로 손상시키거나 삭제하는 행위

정보 보안 기술

방화벽

불법 침입을 차단하고 정상 접근만 허용

사용자 인증

본인 여부를 확인하는 보안 절차

바이러스 백신

악성 소프트웨어를 탐지하고 제거

IDS / IPS

침입 탐지 시스템 / 침입 방지 시스템

블록체인

블록을 분산 저장하고 디지털 서명으로 보안

디지털 윤리

디지털 환경에서 올바른 행동과 가치를 실천하는 것이 중요합니다.
