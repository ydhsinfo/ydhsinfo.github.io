정보 보안의 필요성

Necessity of Information Security

정보 보안이란?

정보를 만들고 사용하는 과정에서 무단 유출, 파괴, 변조 등의 범죄가 일어나지 않도록 방지하는 모든 행위를 말합니다.

정보 보안의 3요소

정보 보안을 유지하기 위한 핵심 3가지 요소

기밀성

허가된 사용자만 정보에 접근할 수 있도록 보장

무결성

허가된 사용자만 정보를 변경할 수 있도록 보장

가용성

허가된 사용자가 필요할 때 정보를 사용할 수 있도록 보장

정보 보안 위협 유형

위조

실제로 존재하지 않는 거짓 정보를 만들어 내는 행위

변조

기존의 정보를 허가 없이 수정하거나 바꾸는 행위

접근과 유출

허가 없이 정보에 접근하거나 외부로 유출하는 행위

훼손과 파괴

정보를 의도적으로 손상시키거나 파괴하는 행위

보안 침해 사례

사물 인터넷에서의 해킹

월패드, IP 카메라 등 사물 인터넷 기기가 해킹되어 집 내부 영상이 외부로 유출되는 사례가 발생하고 있습니다. 네트워크에 연결된 스마트 가전 기기가 보안에 취약한 경우 해커의 공격 대상이 됩니다.

IoT 기기의 초기 비밀번호를 반드시 변경하세요

피싱

이메일, 메신저 등을 통해 신뢰할 수 있는 기관으로 위장하여 개인정보나 금융 정보를 탈취하는 수법입니다. 가짜 웹사이트로 유도하여 로그인 정보를 빼내기도 합니다.

출처가 불분명한 링크는 절대 클릭하지 마세요

스미싱

문자 메시지(SMS)를 이용하여 택배, 청첩장, 공공기관 등을 가장한 메시지를 보내 개인정보를 탈취하거나 소액결제를 유도하는 사기 수법입니다.

의심스러운 문자 메시지의 링크를 열지 마세요

정보 보안 기술

Information Security Technologies

정보 보안을 위해 다양한 기술이 활용됩니다. 각 기술의 역할과 특징을 알아봅시다.

허가받지 않은 사람의 불법적인 침입을 막고, 정상적인 접근만을 허용하는 네트워크 보안 기술입니다. 외부 네트워크와 내부 네트워크 사이에 위치하여 정보의 출입을 통제합니다.

불법 침입 차단 정상 접근 허용 네트워크 보호

사용자의 신원을 확인하여 허가된 사용자만 시스템에 접근할 수 있도록 하는 기술입니다.

지식 기반

패스워드, PIN 번호 등 사용자가 알고 있는 정보

소유 기반

인증서, OTP, 보안 토큰 등 사용자가 소유한 것

생체 기반

지문, 홍채, 얼굴 인식 등 사용자의 고유 생체 정보

네트워크 트래픽을 감시하여 비정상적인 활동을 탐지하고 대응하는 시스템입니다.

IDS (침입 탐지 시스템)

네트워크 침입을 탐지하여 관리자에게 알림을 보내는 시스템

IPS (침입 방지 시스템)

침입을 탐지하고 즉각적으로 방어까지 수행하는 시스템
