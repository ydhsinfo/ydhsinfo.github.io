1

런-렝스 인코딩으로 문자열 압축

문자열 압축 실습

입력 문자열을 런-렝스 인코딩을 이용하여 압축해 보자.

항목 내용
입력 문자열
oopsssss didn't see you
입력 문자열 길이
압축 후 문자열
압축 후 문자열 길이
참고 런-렝스 인코딩 알고리즘으로 문자열 압축하기
01 x = input("압축 전 문자열을 입력하세요: ") # 압축 전 문자열 입력하기
02 result = "" # 압축 후 문자열을 저장할 변수
03 cnt = 1 # 문자 반복 횟수 변수
04 for i in range(1, len(x)) : # i가 1부터 '문자열 길이-1'까지 반복하기
05 if x[i-1] == x[i] : # 문자와 같은 문자이면 반복 횟수 1 증가하기
06 cnt += 1
07 else : # 다음 문자와 같지 않으면
08 if cnt > 1 : # 반복 횟수가 2 이상이면, '횟수 + 문자' 입력
09 result += str(cnt) + x[i-1]
10 else : # 반복 횟수가 1이면, '문자'만 입력
11 result += x[i-1]
12 cnt = 1 # 반복 횟수 1로 초기화하기
13 # 마지막 문자 처리
14 if cnt > 1 :
15 result += str(cnt) + x[i-1]
16 else : # 반복 횟수가 1이면 '문자'만 입력
17 result += x[i]
18 print("압축 후 문자열: ", result) # 압축 후 문자열 출력하기
19 print("압축 전 문자열 길이: ", len(x)) # 압축 전 문자열 길이 출력
20 print("압축 후 문자열 길이: ", len(result)) # 압축 후 문자열 길이 출력

이미지 변환 (BMP → JPG)

이미지 압축 실습

온라인 이미지 변환 사이트에 접속하여 BMP 파일을 JPG 파일로 변환해 보자. 이때 이미지 품질(quality)을 조정해 변환된 이미지와 용량 변화를 확인해 보자.

이미지 변환 사이트 주소

https://convert.io/ko/image-converter

변환 순서

  1. BMP 파일 선택
  2. JPG 형식으로 변환 클릭
  3. 품질(Quality)을 80%, 50%로 각각 조정
  4. 변환 후 파일 크기 비교

확인 포인트

  • BMP 원본과 JPG 변환 후 화질 차이
  • 품질에 따른 파일 크기 변화
  • 손실 압축의 특성 이해
