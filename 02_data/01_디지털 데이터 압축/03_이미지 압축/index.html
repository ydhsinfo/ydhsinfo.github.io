이미지 파일의 다양한 형식과 압축 방식을 이해하고, 손실 압축과 비손실 압축의 차이를 알아봅시다

학습 목표

BMP 파일

Bitmap Image File

BMP 파일이란?

BMP(Bitmap) 파일은 각 화소에 색 정보를 차례대로 기록하는 무압축 파일 형식입니다. 확장자는 .bmp이며, 이미지 품질이 우수하지만 파일 크기가 큰 것이 특징입니다.

화소(Pixel)란?

화소(픽셀, Pixel)는 컴퓨터에서 이미지를 구성하는 최소 단위입니다. 디지털 이미지는 수많은 픽셀이 격자 형태로 모여 하나의 그림을 이루게 됩니다.

2색 표현

각 화소를 0 또는 1로 표현하여 흑백 2색만 나타낼 수 있습니다.

21 = 2색

8비트

256색 표현

8비트로 256가지 색상을 표현할 수 있습니다.

28 = 256색
24

24비트

트루 컬러

RGB 각 8비트로 16,777,216색을 표현하는 트루 컬러입니다.

224 = 16,777,216색

장점

  • 압축하지 않아 이미지 품질이 우수합니다.
  • 원본 데이터가 그대로 유지됩니다.

단점

  • 파일 크기가 매우 큽니다.
  • 웹에서 사용하기에 적합하지 않습니다.

Image Compression Formats

이미지 압축이란?

이미지 데이터를 더 적은 용량으로 저장하는 기술입니다. 비손실 압축은 원본 데이터를 완전히 복원할 수 있고, 손실 압축은 일부 데이터를 제거하여 더 높은 압축률을 얻습니다.

파일 형식별 특징

PNG

비손실 압축

  • 반복되는 색 정보를 더 적은 비트로 줄여 표현합니다.
  • 고품질 이미지 처리에 적합합니다.
  • 투명 배경을 지원합니다.
142 KB 예시 파일 크기

GIF

손실 / 비손실 압축

  • 256색으로 제한하여 파일 크기가 작습니다.
  • 256색 이상 이미지는 손실이 발생합니다.
  • 애니메이션을 지원합니다.
59 KB 예시 파일 크기

JPEG

손실 압축

  • 사람 눈으로 인식하지 못하는 색 차이와 세밀한 부분을 제거합니다.
  • 압축률이 매우 높지만 손실이 발생합니다.
  • 사진 등 자연스러운 이미지에 적합합니다.
8 KB 예시 파일 크기

파일 크기 비교

File Size Comparison

같은 이미지를 다른 파일 형식으로 저장했을 때의 크기를 비교해 봅시다. 압축 방식에 따라 파일 크기가 크게 달라집니다.

BMP
236 KB
PNG
142 KB
GIF
59 KB
JPEG
8 KB

정리

BMP(236KB) > PNG(142KB) > GIF(59KB) > JPEG(8KB) 순으로 파일 크기가 작아집니다. JPEG는 손실 압축을 사용하여 가장 높은 압축률을 보여주지만, 이미지 품질 손실이 발생합니다. PNG는 비손실 압축으로 품질을 유지하면서도 BMP보다 훨씬 작은 크기를 가집니다.

핵심 정리

Key Summary

BMP 파일

  • 각 화소에 색 정보를 차례대로 기록
  • 무압축 파일 형식 (확장자 .bmp)
  • 이미지 품질 우수, 파일 크기 큼

이미지 압축

  • PNG: 비손실 압축, 트루 컬러, 고품질
  • GIF: 256색 제한, 파일 크기 작음, 애니메이션 지원
  • JPEG: 손실 압축, 높은 압축률, 사진에 적합
