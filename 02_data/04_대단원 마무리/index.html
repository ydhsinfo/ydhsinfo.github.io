01

디지털 데이터 압축

데이터 압축: 한정된 저장 공간에 데이터를 효율적으로 기록하기 위한 기술
압축 방식: 손실 압축, 비손실 압축
문자열 압축: 런-렝스 인코딩, 렘펠-지브 인코딩
이미지 압축: BMP, GIF, PNG, JPEG
소리 압축: WAV, FLAC, MP3, AAC
02

디지털 데이터 암호화

데이터 암호화: 권한이 없는 사람이 데이터를 알아볼 수 없게 변환하는 기술
핵심 개념: 암호화와 복호화, 평문 / 암호문 / 암호 알고리즘 / 키
고대 · 근대 암호: 전치 암호, 치환 암호
현대 암호: 컴퓨터 프로그램 기반 암호 기술
03

빅데이터 개념과 분석

빅데이터: 대규모 데이터를 저장 · 분석하여 가치를 추출
활용 범위: 사회 전 영역에서 활용
분석 4단계: 문제 정의 → 데이터 수집 → 데이터 분석 → 공유 · 평가

실력 확인

배운 내용을 문제로 확인해 봅시다

문제 1 주관식

데이터 압축에서 원래의 데이터를 더 작은 크기로 변환하는 (㉠) 과정과 압축된 데이터를 원래의 데이터로 복원하는 (㉡) 과정이 있다. ㉠과 ㉡에 들어갈 용어를 순서대로 쓰시오.

문제 2 객관식

디지털 데이터 압축에 대한 설명 중 옳은 것은?

문제 3 객관식

사람의 눈이 인식하지 못하는 범위의 정보를 제거하여 이미지 크기를 줄인 형식은?

문제 4 객관식

권한이 없는 사람이 데이터에 접근하지 못하도록 데이터를 변환하는 기술은?

문제 5 객관식

암호 알고리즘에서 평문을 암호문으로 변경할 때 사용하는 문자열은?

문제 6 객관식

평문의 알파벳을 키만큼 평행 이동한 알파벳으로 치환하는 암호는?

문제 7 객관식

다음 설명에 해당하는 암호 알고리즘을 바르게 짝지은 것은?

(가) 데이터를 고정된 길이의 값으로 변환하며, 원본 복원이 불가능

(나) 암호화와 복호화에 서로 다른 키를 사용

(다) 암호화와 복호화에 같은 키를 사용

문제 8 객관식

다음 중 데이터 유형이 다른 하나는?

문제 9 서술형

대규모 데이터를 저장 · 분석하여 가치를 추출하고 다양한 분야에서 활용하는 개념을 무엇이라 하는가?

문제 10 객관식

빅데이터 분석 과정을 올바른 순서로 나열한 것은?

ㄱ. 데이터 분석 ㄴ. 데이터 수집 ㄷ. 문제 정의 ㄹ. 공유 · 평가
