1

문제 정의하기

분석의 출발점

빅데이터 분석의 첫 단계는 분석의 필요성, 요구 사항, 분석 범위 등을 정확히 파악하는 것입니다. 무엇을 알고 싶은지, 어떤 문제를 해결하고자 하는지를 명확히 해야 합니다.

파악해야 할 사항

  • 분석의 필요성 확인
  • 구체적인 요구 사항 정리
  • 분석 범위 설정

핵심 활동

구체적인 문제의 분석 목적을 정의합니다. 모호한 목적은 잘못된 방향으로 분석을 이끌 수 있으므로, 명확하고 측정 가능한 목적을 세우는 것이 중요합니다.

예시: 문제 정의

"우리 학교 학생들의 통학 시간이 학업 성취도에 영향을 미치는가?"

필요성: 통학 부담 개선 범위: 우리 학교 전교생 목적: 상관관계 분석
2

데이터 수집하기

가장 중요한 단계

데이터 수집은 분석 결과에 가장 큰 영향을 미치는 단계입니다. 잘못된 데이터를 수집하면 아무리 좋은 분석 기법을 사용해도 의미 있는 결과를 얻을 수 없습니다.

데이터 수집 시 고려사항

데이터양의 균형

수집하는 데이터의 양이 너무 적으면 신뢰성이 떨어지고, 너무 많으면 처리가 어렵습니다.

개인 정보 노출

데이터 수집 시 개인 정보가 포함되지 않도록 주의하고, 관련 법규를 준수해야 합니다.

데이터 편향성

특정 집단에 치우친 데이터는 분석 결과를 왜곡할 수 있으므로 균형 잡힌 수집이 필요합니다.

저작권

타인이 만든 데이터를 사용할 때는 저작권과 이용 조건을 확인해야 합니다.

데이터 수집 방법

자동화된 수집기

웹 크롤러, API 등을 이용한 자동 수집

인터넷 직접 검색

포털 사이트나 데이터 제공 사이트를 통한 수집

직접 수집

설문 조사, 센서 측정, 관찰 기록 등

누락된 값으로, 데이터 수집 과정에서 빠진 항목입니다. 삭제하거나 대표값으로 대체합니다.

2 데이터 시각화 · 분석하기

전처리된 데이터를 목적에 맞는 차트로 시각화하여 의미를 파악합니다.

시간 시각화

Time Visualization

시간에 따른 데이터 추이를 파악할 때 사용합니다.

선 그래프 (Line Chart)
막대 그래프 (Bar Chart)
영역형 차트 (Area Chart)

활용 예시: 연도별 인구 변화, 월별 매출 추이, 일별 기온 변화 등 시간의 흐름에 따라 데이터가 어떻게 변하는지 확인할 수 있습니다.

관계 시각화

Relationship Visualization

데이터 간 연관성을 파악할 때 사용합니다.

버블 차트 (Bubble Chart)
히스토그램 (Histogram)
산점도 (Scatter Plot)

활용 예시: 키와 몸무게의 관계, 학습 시간과 성적의 상관관계, 기온과 아이스크림 판매량의 연관성 등을 분석할 수 있습니다.

비교 시각화

Comparison Visualization

데이터 간 차이와 유사성을 비교할 때 사용합니다.

히트맵 (Heatmap)
워드 클라우드 (Word Cloud)
방사형 차트 (Radar Chart)

활용 예시: 지역별 기온 분포(히트맵), 뉴스 키워드 빈도(워드 클라우드), 학생 역량 비교(방사형 차트) 등에 활용됩니다.

4

공유 및 평가하기

분석의 마무리

분석 결과를 다른 사람들과 공유하고, 그 결과를 검토하고 평가하는 마지막 단계입니다.

분석 결과 검토 및 평가

  • 분석 결과가 초기 문제 정의와 부합하는지 확인
  • 데이터와 분석 방법의 적절성 평가
  • 결과의 신뢰성과 타당성 검증

개선책/해결책 제시

  • 분석 결과에 기반한 올바른 개선책 도출
  • 실행 가능한 해결책 제시
  • 이해관계자와 결과 공유 및 피드백 수렴

기억하세요!

빅데이터 분석은 문제 정의 → 데이터 수집 → 데이터 분석 → 공유 및 평가의 순환적 과정입니다. 평가 결과에 따라 이전 단계로 되돌아가 분석을 보완할 수도 있습니다. 이 과정에서 가장 중요한 것은 정확한 문제 정의와 신뢰할 수 있는 데이터 수집입니다.

핵심 정리

빅데이터 분석 과정

문제 정의 → 데이터 수집 → 데이터 분석(전처리 + 시각화) → 공유 및 평가

수집 시 고려사항

데이터양의 균형, 개인 정보 보호, 데이터 편향성, 저작권 준수

전처리 핵심

평균/중앙값/최빈값 파악, 결측치/중복값/이상치 처리, 박스 플롯 활용

시각화 유형

시간 시각화(추이), 관계 시각화(연관성), 비교 시각화(차이점)
