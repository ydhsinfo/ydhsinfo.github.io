빅데이터 개념과 분석

빅데이터 개념 이해

디지털 환경에서 생성되는 대규모 데이터의 유형과 특징을 이해하고, 빅데이터가 가져오는 새로운 가치를 탐구합니다.

학습 목표

  • 빅데이터의 개념을 설명하고, 정형 데이터와 비정형 데이터를 구분할 수 있다.
  • 빅데이터의 5V 특징(규모, 속도, 다양성, 정확성, 가치)을 이해하고 설명할 수 있다.
  • 디지털 환경에서 데이터가 생산되는 규모와 속도를 인식한다.
1

빅데이터의 데이터 유형

빅데이터란?

빅데이터(Big Data)란 디지털 환경에서 생성된 수십 테라바이트(TB) 이상의 대규모 데이터를 말합니다. 다양한 유형의 대규모 데이터를 효과적으로 저장하고, 통계, 컴퓨터 과학, 기계학습 등의 데이터 과학 기술로 가치를 추출하는 것이 핵심입니다.

IoT(사물인터넷)와 스마트 기기의 확산으로 데이터를 생산하는 사물의 범위가 기하급수적으로 증가하고 있습니다. 스마트폰, 웨어러블 기기, 자율주행 자동차, 스마트 가전 등 거의 모든 디지털 기기가 데이터를 생산합니다.

알고 계셨나요?

전 세계에서 생성되는 데이터의 약 80~90%가 비정형 데이터입니다. 빅데이터 기술의 발전으로 이러한 비정형 데이터에서도 의미 있는 가치를 추출할 수 있게 되었습니다.

2

빅데이터의 특징 (5V)

빅데이터의 특징은 흔히 5V로 설명합니다. 각각의 V는 빅데이터를 정의하는 핵심 속성을 나타냅니다. 아래 원형 다이어그램의 각 항목을 클릭하여 자세한 내용을 확인하세요.

5V
  • Volume (규모): 데이터의 규모가 크다
  • Velocity (속도): 데이터 생성, 수집, 처리 속도가 빠르다
  • Variety (다양성): 다양한 유형의 데이터를 처리
  • Veracity (정확성): 유용하고 신뢰할 수 있는 데이터
  • Value (가치): 숨겨진 가치를 찾을 수 있다

왼쪽 다이어그램의 항목을 클릭하세요

규모
Volume

데이터의 규모가 크다. 수십 테라바이트(TB)에서 페타바이트(PB) 이상의 방대한 데이터를 다룹니다.

속도
Velocity

데이터의 생성, 수집, 처리 속도가 빠르다. 실시간으로 쏟아지는 데이터를 신속하게 처리해야 합니다.

다양성
Variety

다양한 유형(정형, 비정형)의 데이터를 처리합니다. 텍스트, 이미지, 동영상, 센서 데이터 등 형식이 다양합니다.

정확성
Veracity

유용하고 신뢰할 수 있는 데이터를 다룹니다. 데이터의 품질과 정확성을 확보하는 것이 중요합니다.

가치
Value

숨겨진 가치를 찾을 수 있다. 대량의 데이터 분석을 통해 새로운 통찰과 비즈니스 가치를 창출합니다.

핵심 정리

빅데이터

디지털 환경에서 생성된 수십 TB 이상의 대규모 데이터. 데이터 과학 기술로 가치를 추출하는 것이 목적입니다.

데이터 유형

정형 데이터(표, CSV)와 비정형 데이터(이미지, 소리, 문서)로 구분됩니다.

5V 특징

Volume(규모), Velocity(속도), Variety(다양성), Veracity(정확성), Value(가치)

데이터 폭증

IoT와 스마트 기기 확산으로 데이터 생산량이 기하급수적으로 증가하고 있습니다.
