3

빅데이터 분석과 활용 문제점

빅데이터 활용 시 발생할 수 있는 문제점을 탐구합니다

아래 사례를 읽고, 각 사례에서 발생하는 문제점을 파악한 뒤 해결 방안을 작성해 봅시다.

1 맞춤형 광고와 프라이버시 침해

인터넷 검색 기록, 구매 이력, 위치 정보 등을 분석하여 개인 맞춤형 광고를 제공하지만, 이 과정에서 개인의 프라이버시가 침해될 수 있습니다.
프라이버시 침해 - 개인의 검색 기록, 구매 이력, 위치 정보 등 사적 데이터가 동의 없이 수집/활용되어 사생활이 침해됨

2 아이스크림 판매량과 익사 사고 - 인과관계 확대 해석

아이스크림 판매량이 늘어나면 익사 사고도 늘어나는 상관관계가 있지만, 이것을 "아이스크림이 익사를 유발한다"라고 해석하는 것은 인과관계를 확대 해석한 것입니다. 실제 원인은 '기온 상승'이라는 제3의 변수입니다.
인과관계 확대 해석 - 상관관계가 있는 데이터를 인과관계로 잘못 해석하여 잘못된 결론을 도출할 수 있음

3 신용카드 분석과 데이터 속성의 다중 의미

신용카드 사용 내역을 분석하면 소비 패턴을 파악할 수 있지만, 동일한 데이터도 맥락에 따라 다양하게 해석될 수 있습니다. 예를 들어, 특정 시간대의 편의점 결제가 야근인지 야식 습관인지 구분하기 어렵습니다.
데이터 속성의 다중 의미 - 하나의 데이터가 여러 의미로 해석될 수 있어 분석 결과가 왜곡될 수 있음
