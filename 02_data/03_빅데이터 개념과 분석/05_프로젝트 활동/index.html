스마트폰 센서

스마트폰은 동작, 환경, 위치 관련 센서로 기기의 움직임과 위치 정보를 모니터링하고 주변 환경의 변화를 감지한다.

스마트폰 센서 종류

다양한 센서의 역할과 기능

제스처 센서

손의 움직임을 감지하여 터치 없이 조작 가능

RGB 센서

빛의 삼원색(Red, Green, Blue)을 측정

홀 센서

자기장 감지로 커버 덮임 여부 확인

기압 센서

기압 변화를 감지하여 높이(고도) 측정

온습도 센서

주위 환경의 온도와 습도 측정

근접 센서

물체 간의 거리를 측정 (통화 시 화면 끄기)

자이로 센서

회전 상태와 기울기를 감지

실습 사용

가속도 센서

이동 상태(속도 변화)를 감지

지자기 센서

자기장 세기를 측정하여 방위(나침반) 기능

단계 2: 데이터 수집하기

센서 앱으로 데이터 수집

Physics Toolbox Sensor Suite

스마트폰 센서 데이터 수집 앱

이 앱을 사용하면 스마트폰의 다양한 센서 데이터를 실시간으로 측정하고 CSV 파일로 저장할 수 있습니다.

데이터 수집 절차

1

앱 설치 및 실행

Physics Toolbox Sensor Suite 앱을 스마트폰에 설치하고 실행합니다.

2

리니어 가속도 선택

센서 목록에서 리니어 가속도(Linear Acceleration) 측정을 선택합니다.

3

설정 조정

X축을 경과 시간(Elapsed Time)으로 설정하고, 측정 속도를 Slowest로 설정합니다.

4

운동하며 데이터 수집

운동을 시작할 때 저장 버튼을 누르고, 걷기/달리기/정지 등 다양한 활동을 수행합니다.

5

CSV 파일로 저장

운동이 끝나면 저장을 중지하고, 수집된 데이터를 CSV 파일로 저장합니다.

팁 — 걷기, 달리기, 정지 등 다양한 활동을 번갈아 수행하면 분석 결과가 더 명확하게 나타납니다. 각 활동 사이에 잠시 멈추는 구간을 넣어 보세요.
3

단계 3: 데이터 분석하기

코답(CODAP)으로 시각화 및 분석

코답(CODAP)

https://codap.concord.org/

코답(CODAP)은 웹 기반 데이터 분석 도구로, CSV 파일을 불러와 다양한 그래프를 손쉽게 만들 수 있습니다.

분석 방법

1

코답 사이트에 접속하여 수집한 CSV 파일을 불러옵니다.

2

시간에 따른 총 가속도(atotal) 변화를 선그래프로 나타냅니다.

3

그래프의 패턴을 관찰하여 걷기, 달리기, 정지 구간을 구별합니다.

가속도 데이터 해석 가이드

달리는 중

가속도 높음

가속도 값의 변동 폭이 크고 높은 값을 나타냄

걷는 중

가속도 낮음

가속도 값의 변동 폭이 작고 낮은 값을 나타냄

움직임 없음

가속도 0

가속도 값이 거의 0에 가까운 상태

선그래프 해석 예시

0s 시간 (경과 시간) 끝
정지
걷기
달리기

Y축: 총 가속도(atotal)  |  X축: 경과 시간

단계 4: 공유 및 평가

결과 해석 및 토론

분석 결과 공유

  • 친구들과 분석 결과를 비교하고 해석해 봅니다.
  • 각자의 운동 활동 패턴에서 공통점과 차이점을 찾아봅니다.
  • 그래프에서 특이한 구간이 있다면 원인을 분석해 봅니다.

효과적인 운동 방법

  • 데이터를 기반으로 효과적인 운동 방법에 대해 이야기해 봅니다.
  • 운동 강도와 가속도의 관계를 논의합니다.
  • 센서 데이터 활용의 다른 가능성을 탐색합니다.
