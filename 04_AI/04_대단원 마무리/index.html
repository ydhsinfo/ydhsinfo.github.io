개념 정리 실력 확인

개념 정리

Key Concepts Summary

01 인공지능과 지능 에이전트

인공지능

사람의 지능적인 행위나 판단, 행동 등을 컴퓨터로 구현한 기술

지능 에이전트

주변 환경과 상호 작용하면서 지능적인 판단과 결정 과정에서 자율적으로 행동하는 시스템

02 기계학습의 개념

기계학습이란?

수많은 데이터로 학습하여 스스로 규칙을 발견하는 인공지능의 한 분야

지도학습

입력 데이터 + 정답을 함께 학습하여 새로운 데이터를 예측

비지도학습

정답 없이 데이터의 패턴과 구조를 스스로 발견

강화 학습

보상 경험을 통해 최적의 행동 전략을 학습

03 기계학습의 모델과 구현

회귀

연속적 수치형 데이터를 예측하는 모델

분류

둘 이상의 범주 중 어디에 해당하는지 예측

kNN 결정트리 로지스틱 회귀

군집화

비슷한 특징끼리 그룹으로 묶는 모델

연관 규칙

데이터 항목 간 연관성을 분석하는 모델

기계학습 구현 과정

STEP 1
문제 정의
STEP 2
데이터 수집/전처리
STEP 3
모델 선정/학습
STEP 4
모델 평가
