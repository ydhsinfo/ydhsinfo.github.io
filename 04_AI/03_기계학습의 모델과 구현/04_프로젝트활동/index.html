단계 5: 모델 활용 및 배포

Model Deployment & Application

티처블머신에서 학습한 모델을 클라우드에 업로드하고, 공유 가능한 링크를 생성하여 다른 사람들과 공유할 수 있습니다.

Keras / TensorFlow로 내보내기

학습된 모델을 Keras 또는 TensorFlow 형식으로 내보내어 Python 환경에서 활용할 수 있습니다.

Python 코드로 활용 (Jupyter Notebook)

내보낸 모델을 Python에서 실행하기

# 필요한 라이브러리 설치
# pip install tensorflow pillow numpy

from tensorflow.keras.models import load_model
from PIL import Image
import numpy as np

# 모델 로드
model = load_model("keras_model.h5")
labels = ["유리", "종이", "플라스틱"]

# 이미지 전처리
image = Image.open("test_image.jpg")
image = image.resize((224, 224))
img_array = np.array(image) / 255.0
img_array = np.expand_dims(img_array, axis=0)

# 예측
prediction = model.predict(img_array)
result = labels[np.argmax(prediction)]
print(f"분류 결과: {result}")
참고: Jupyter Notebook을 사용하면 코드를 단계별로 실행하며 결과를 바로 확인할 수 있습니다.
