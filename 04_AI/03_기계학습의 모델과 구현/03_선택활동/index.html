모델별 결과 이해하기

Understanding Model Results

k-최근접 이웃 (k-Nearest Neighbors)

새로운 데이터를 가장 가까운 k개의 이웃 데이터를 기준으로 분류

핵심 파라미터: K값

  • K=3: 가장 가까운 3개의 이웃을 참조하여 다수결로 분류. 경계가 복잡하고 노이즈에 민감
  • K=5: 가장 가까운 5개의 이웃을 참조. 경계가 부드럽고 안정적인 분류 결과

관찰 포인트

  • K값이 작을수록 경계가 울퉁불퉁(과적합 가능성)
  • K값이 클수록 경계가 부드러움(일반화 향상)
  • 겹치는 영역에서의 분류 결과 차이를 주목

결정트리 (Decision Tree)

데이터를 조건에 따라 나무 구조로 분할하여 분류

핵심 파라미터: Max Tree Depth

  • Depth=3: 트리의 깊이를 3으로 제한. 단순한 분류 경계를 생성하며 일반화 능력이 좋음
  • Depth=5: 트리의 깊이를 5로 확장. 복잡한 경계를 형성하며 학습 데이터에 더 잘 맞음

관찰 포인트

  • 경계선이 직각(수평/수직)으로 나뉘는 특징
  • 깊이가 깊을수록 경계가 복잡해짐(과적합 위험)
  • 직관적이고 해석이 쉬운 모델 구조

인공 신경망 (Neural Network)

뉴런의 연결 구조를 모방하여 비선형 패턴을 학습

핵심 특징

  • 입력층, 은닉층, 출력층으로 구성된 다층 구조
  • 반복 학습(에포크)을 통해 가중치를 점진적으로 조정
  • 비선형 활성화 함수로 복잡한 패턴 학습 가능

관찰 포인트

  • 부드러운 곡선 형태의 분류 경계를 생성
  • 복잡한 데이터 분포에서도 우수한 분류 성능
  • 학습 시간이 다른 모델에 비해 상대적으로 오래 걸림

KNN 상세 가이드

적합한 경우

  • 데이터 양이 적을 때
  • 별도의 학습 과정 없이 빠르게 결과를 확인하고 싶을 때
  • 데이터 분포가 명확하게 구분될 때

주의할 점

  • 데이터 양이 많으면 예측 속도가 느림
  • 고차원 데이터에서 성능 저하
  • K값 선택이 결과에 큰 영향

결정트리 상세 가이드

적합한 경우

  • 결과의 이유를 설명해야 할 때
  • 범주형과 수치형 데이터가 혼합되어 있을 때
  • 빠른 학습과 예측이 필요할 때

주의할 점

  • 깊이가 너무 깊으면 과적합 발생
  • 작은 데이터 변화에 민감하게 반응
  • 직선 경계만 생성하여 복잡한 패턴에 한계

인공 신경망 상세 가이드

적합한 경우

  • 복잡한 비선형 패턴을 학습해야 할 때
  • 대용량 데이터가 충분히 확보될 때
  • 이미지, 음성 등 비정형 데이터를 다룰 때

주의할 점

  • 학습 시간이 오래 걸림
  • 결과 해석이 어려움(블랙박스)
  • 많은 하이퍼파라미터 조정 필요

자기 평가

Self Evaluation

기계학습의 모델별 개념을 이해할 수 있는가?

매우 잘 이해함

KNN, 결정트리, 인공 신경망의 개념과 차이점을 명확하게 설명할 수 있다.

대체로 이해함

각 모델의 기본 개념은 이해하지만, 세부적인 차이점 설명에 어려움이 있다.

추가 학습 필요

모델별 개념이 아직 혼동되며, 반복 학습이 필요하다.

핵심 정리

Key Concepts Summary

KNN

  • 가장 가까운 K개의 이웃으로 다수결 분류
  • K값이 작으면 복잡, 크면 단순한 경계
  • 별도 학습 과정 없이 바로 예측

결정트리

  • 조건에 따라 나무 구조로 분류
  • 깊이(Depth)가 깊을수록 복잡한 분류
  • 직관적이고 해석이 쉬운 모델

인공 신경망

  • 뉴런 연결 구조를 모방한 학습
  • 복잡한 비선형 패턴 학습 가능
  • 부드러운 곡선 분류 경계 생성
