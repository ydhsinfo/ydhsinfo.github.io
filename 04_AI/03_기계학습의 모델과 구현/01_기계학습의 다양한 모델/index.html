기계학습의 다양한 모델

회귀, 분류, 군집화, 연관 규칙 등 기계학습의 주요 모델을 이해하고, 각 모델의 특징과 활용 사례를 탐구합니다

회귀 분류 군집화 연관 규칙

학습 목표

1기계학습의 다양한 모델(회귀, 분류, 군집화, 연관 규칙)의 개념과 특징을 이해할 수 있다.
2지도학습과 비지도학습의 차이를 설명하고, 각 모델의 실생활 활용 사례를 분석할 수 있다.
회귀 분류 군집화 연관 규칙

기계학습 모델 체계

Machine Learning Model Hierarchy

기계학습은 크게 지도학습, 비지도학습, 강화학습으로 분류됩니다

지도학습

입력 데이터와 정답(레이블)이 함께 주어진 상태에서 학습합니다. 새로운 데이터의 정답을 예측합니다.

회귀분류

비지도학습

정답(레이블) 없이 데이터 자체에서 패턴이나 구조를 스스로 발견합니다.

군집화연관 규칙

회귀 모델

Regression Model

회귀(Regression)란?

데이터의 패턴을 학습하여 연속적인 수치형 데이터를 예측하는 지도학습 모델입니다. 입력 변수와 출력 변수 사이의 관계를 수학적 함수로 나타냅니다.

예시: 학습 시간과 성적 예측 (선형 회귀)

학습 시간이 늘어날수록 성적이 높아지는 선형 관계를 회귀선으로 나타내어 새로운 학습 시간에 대한 성적을 예측합니다.

단순 선형 회귀

하나의 입력 변수로 결과 예측
예: 학습 시간 → 성적

다중 선형 회귀

여러 입력 변수로 결과 예측
예: 학습 시간 + 출석률 + 과제 → 성적

회귀 모델의 활용 사례

강우량 예측

기온, 습도, 기압 데이터로 강우량을 예측

중고차 가격 예측

연식, 주행거리, 차종으로 가격을 예측

원유 가격 예측

수요, 공급, 환율 등으로 원유 가격 예측

지하철 승객 예측

시간대, 요일, 날씨로 탑승 인원 예측

분류 모델

Classification Model

분류(Classification)란?

입력 데이터가 어떤 범주(클래스)에 해당하는지 판단하는 지도학습 모델입니다.

분류 알고리즘의 종류

1결정 트리 (Decision Tree)

데이터를 트리(나무) 구조로 분류합니다. 각 노드에서 특정 기준에 따라 데이터를 나누어 최종 분류 결과를 결정합니다.

위조 지폐 판별 예시

크기, 무게, 인쇄 품질 등을 기준으로 트리를 따라가며 진짜/위조를 판별합니다.

2로지스틱 회귀 (Logistic Regression)

입력 데이터를 0과 1 사이의 확률값으로 변환하여 분류합니다. 이름에 "회귀"가 포함되지만 실제로는 분류에 사용됩니다.

분류 기준

예측 확률이 0.5 이상이면 양성(1), 0.5 미만이면 음성(0)으로 분류합니다.

활용 예시

이메일 스팸 판별, 질병 진단(양성/음성), 합격/불합격 예측 등

3k-최근접 이웃 (kNN)

새로운 데이터가 입력되면 가장 가까운 k개의 이웃 데이터를 확인하여, 그 이웃들 중 가장 많은 클래스로 분류합니다.

k값의 중요성

k=3이면 가장 가까운 3개 이웃을 확인하여 다수결로 결정합니다. k값에 따라 결과가 달라질 수 있습니다.

4인공 신경망 (ANN)

사람의 뇌 신경세포(뉴런)를 모방한 알고리즘입니다. 입력층, 은닉층, 출력층으로 구성되어 데이터가 층을 통과하며 학습합니다.

심층 신경망 (DNN)

은닉층이 2개 이상인 인공 신경망을 심층 신경망이라 하며, 이를 사용한 학습을 딥러닝(Deep Learning)이라고 합니다.

군집화 (비지도학습)

Clustering

군집화(Clustering)란?

정답 없이 비슷한 특징을 가진 데이터끼리 그룹(군집)으로 묶는 비지도학습 모델입니다.

k-평균 군집화 알고리즘

데이터를 k개의 군집으로 나누는 대표적인 군집화 알고리즘입니다.

1

k개의 중심점을
임의로 설정

2

각 데이터를 가장
가까운 중심점에 배정

3

각 군집의 평균으로
중심점을 재계산

4

변화가 없을 때까지
2~3단계 반복

셔츠 크기 군집화

비지도학습

키와 몸무게 데이터를 기반으로 고객들을 S, M, L, XL 등의 그룹으로 자동 분류하여 최적의 사이즈 체계를 만듭니다.

화장품 브랜드 군집화

비지도학습

구매 패턴, 연령대, 선호도를 분석하여 유사한 고객 그룹을 발견하고 맞춤 마케팅 전략을 수립합니다.

연관 규칙 (비지도학습)

Association Rules

연관 규칙(Association Rules)이란?

대량의 데이터에서 항목들 간의 연관성(관계)을 찾아내는 비지도학습 모델입니다.

장바구니 분석 예시

우유 구매

식빵도 구매

"우유를 구매한 고객은 식빵도 함께 구매하는 경향이 있다"

고객의 구매 이력을 분석하여 함께 구매되는 상품의 연관성을 찾고, 상품 배치, 묶음 할인, 추천 서비스 등에 활용합니다.

연관 규칙 기반 추천 서비스

음악 추천

비슷한 취향의 사용자가 들은 곡을 기반으로 새로운 음악을 추천

도서 추천

같은 책을 읽은 독자들이 함께 읽은 다른 책을 추천

웹툰 추천

비슷한 장르나 작가의 작품을 연관 분석하여 추천

핵심 정리

회귀 모델

  • 연속적인 수치형 데이터를 예측하는 지도학습
  • 단순 선형 회귀, 다중 선형 회귀

분류 모델

  • 데이터를 범주(클래스)로 판별하는 지도학습
  • 결정 트리, 로지스틱 회귀, kNN, 인공 신경망

군집화

  • 비슷한 데이터끼리 그룹화하는 비지도학습
  • k-평균 군집화 알고리즘이 대표적

연관 규칙

  • 항목 간 연관성을 찾는 비지도학습
  • 추천 시스템, 장바구니 분석 등에 활용
