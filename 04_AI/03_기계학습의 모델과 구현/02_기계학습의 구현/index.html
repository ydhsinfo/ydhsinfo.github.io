기계학습의 구현 과정

Machine Learning Implementation Process

기계학습 구현이란?

문제 정의 → 데이터 수집 및 전처리 → 모델 선정 및 학습 → 모델 평가의 4단계 과정을 거쳐 기계학습을 구현합니다.

1

문제 정의하기

해결하고자 하는 문제가 기계학습으로 해결 가능한지 판단합니다.

2

데이터 수집 및 전처리

통계 정보를 확인하고, 결측치 및 이상치를 처리합니다.

3

모델 선정 및 학습

분류/회귀/군집화 중 적합한 모델을 선택하고 훈련/테스트 데이터로 학습합니다.

4

모델 평가하기

학습 결과를 평가하고, 성능이 부족하면 모델을 수정하거나 변경합니다.

구현 과정 흐름도

다양한 의류 이미지를 종류별로 자동 분류하여 관리합니다.

오렌지3(Orange3) 소개

Introduction to Orange3

오렌지3(Orange3)란?

오픈 소스 기계학습 도구로, 코드를 작성하지 않고 위젯을 드래그 앤 드롭하여 기계학습 워크플로를 구성할 수 있습니다.

Data

데이터 불러오기, 저장, 정보 확인

Transform

데이터 변환, 필터링, 전처리

Visualize

데이터를 차트/그래프로 시각화

Model

기계학습 모델 학습 (회귀, 분류 등)

Evaluate

모델 성능 테스트 및 평가

Unsupervised

비지도 학습(군집화 등)

워크플로 구성 방법

1

위젯 선택 후 캔버스에 배치

2

위젯 간 선으로 연결

3

위젯 더블클릭으로 설정/결과 확인

실습: 지구 온도 변화 예측

Practice: Global Temperature Prediction (Regression)

회귀(Regression) 문제

과거의 지구 평균 온도 데이터를 학습하여 미래의 온도 변화를 예측하는 회귀 문제입니다.

  • 캐글에서 CSV 데이터 다운로드 → File 위젯으로 불러오기
  • Data Table로 통계 정보 확인, Preprocess로 결측치/이상치 처리

연속적인 값을 예측하는 회귀 문제이므로, Linear Regression(선형 회귀) 모델을 사용합니다.

File → Data Table → Select Columns → Linear Regression → Predictions

Test & Score 위젯을 사용하여 모델의 성능을 평가합니다.

회귀 모델 평가 지표

지표 의미 좋은 값
MSE평균 제곱 오차0에 가까울수록 좋음
RMSE평균 제곱근 오차0에 가까울수록 좋음
MAE평균 절대 오차0에 가까울수록 좋음
R2결정 계수 (모델의 설명력)1에 가까울수록 좋음

분류(Classification) 문제

아델리 펭귄

Adelie

남극 해안 전역에 서식하는 가장 널리 분포된 펭귄 종

턱끈 펭귄

Chinstrap

턱 아래의 검은 줄무늬가 특징인 펭귄 종

젠투 펭귄

Gentoo

머리 위의 흰색 띠가 특징이며 가장 빠르게 헤엄치는 펭귄

분류 모델 비교

k-최근접 이웃 (kNN)

새로운 데이터가 입력되면, 가장 가까운 k개의 이웃 데이터를 찾아 다수결로 분류합니다.

결정트리 (Decision Tree)

데이터의 특성값을 기준으로 질문을 반복하며 트리 구조로 분류합니다. 해석이 용이합니다.

로지스틱 회귀 (Logistic Regression)

데이터가 특정 클래스에 속할 확률을 계산하여 분류합니다. 이름은 회귀이지만 분류에 사용됩니다.

CA (Classification Accuracy) - 분류 정확도

전체 데이터 중 올바르게 분류된 데이터의 비율입니다. 1에 가까울수록 우수한 모델입니다. 오렌지3의 Test & Score 위젯에서 각 모델의 CA를 비교합니다.

펭귄 분류 실습 워크플로

3가지 분류 모델의 CA(분류 정확도)를 비교하여 가장 좋은 모델을 선택합니다

핵심 정리

구현 과정 4단계

  • 문제 정의 → 데이터 수집/전처리 → 모델 선정/학습 → 평가
  • 결측치, 이상치 처리는 데이터 전처리 단계에서 수행
  • 평가 결과가 부족하면 이전 단계로 돌아가 반복

오렌지3(Orange3)

  • 코드 없이 위젯 드래그 앤 드롭으로 기계학습 구현
  • Data, Transform, Visualize, Model, Evaluate, Unsupervised

실습 정리

지구 온도 변화 예측 (회귀)

Linear Regression, MSE/RMSE/MAE/R2로 평가

펭귄 종 분류 (분류)

kNN, 결정트리, 로지스틱 회귀 비교, CA로 평가
