기계학습의 유형

기계학습의 세 가지 유형인 지도학습, 비지도학습, 강화학습의 개념과 활용 사례를 탐구합니다

지도학습 비지도학습 강화 학습

학습 목표

1 기계학습의 세 가지 유형(지도학습, 비지도학습, 강화학습)의 개념과 차이점을 설명할 수 있다.
2 각 기계학습 유형의 실생활 활용 사례를 분석하고 적용할 수 있다.
지도학습 비지도학습 강화 학습

지도학습

Supervised Learning

지도학습이란?

입력 데이터와 그에 대응하는 정답 데이터(레이블)를 함께 제공하여 데이터 간의 관계와 패턴을 학습하는 방법입니다. 선생님이 문제와 정답을 알려주며 가르치는 것과 같습니다.

특징(Feature)

모델이 학습에 사용하는 입력 데이터의 속성입니다. 동물 분류에서 다리 수, 크기 등이 특징에 해당합니다.

레이블(Label)

각 입력 데이터에 대한 정답(결과값)입니다. 모델은 특징과 레이블의 관계를 학습하여 새로운 데이터의 결과를 예측합니다.

예시: 거북이와 물고기 분류

비지도학습이란?

강화학습

Reinforcement Learning

강화학습이란?

에이전트가 환경과 상호 작용하며 시행착오를 통해 점점 나은 행동을 학습합니다. 좋은 행동에는 보상을, 나쁜 행동에는 벌칙을 주어 보상을 최대화하는 전략을 학습합니다.

강화학습의 구조

로봇 보행

수천 번의 시뮬레이션으로 걷는 방법을 스스로 학습

보상: 멀리 걸을수록 +

벌칙: 넘어지면 -

게임 AI

게임 캐릭터가 보상을 최대화하도록 최적 전략 학습

보상: 점수 획득

벌칙: 게임 오버

알파고 (AlphaGo)

수백만 번의 연습 경기로 세계 챔피언을 이긴 바둑 AI

보상: 승리 +1

벌칙: 패배 -1

지도학습 (Supervised Learning)

입력과 정답(레이블)을 함께 제공하여 학습합니다. 선생님이 문제와 답을 알려주는 것처럼 데이터 간의 관계를 배워 새로운 데이터의 결과를 예측합니다.

날씨 예측 스팸 분류 질병 진단

비지도학습 (Unsupervised Learning)

정답 없이 데이터 속에서 스스로 패턴과 구조를 발견합니다. 비슷한 데이터를 그룹으로 묶거나 숨겨진 특징을 추출합니다.

이상 탐지 고객 세분화 도서 추천

강화학습 (Reinforcement Learning)

시행착오를 통해 보상을 최대화하는 행동을 학습합니다. 에이전트가 환경과 상호 작용하며 최적의 전략을 스스로 터득합니다.

핵심 정리

지도학습

  • 입력 + 정답(레이블)으로 학습
  • 특징과 레이블 관계 파악
  • 예측, 분류 문제에 활용

비지도학습

  • 정답 없이 스스로 학습
  • 데이터의 패턴/구조 발견
  • 군집화, 이상 탐지에 활용

강화학습

  • 시행착오로 보상 최대화
  • 에이전트-환경 상호 작용
  • 게임 AI, 로봇 제어에 활용
